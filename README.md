
# Task Tracker

Simple React todo/task tracker app for adding, editing and filtering tasks.

Features:
- Add, edit, and delete tasks
- Mark tasks complete/incomplete
- Filter tasks by status and search

Getting Started

Prerequisites:
- Node.js (16+) and npm or yarn

Install:
```bash
npm install
```

Run (development):
```bash
npm run dev
```

Build:
```bash
npm run build
```

Project Structure
- `src/` — React components, hooks, services, and assets
- `public/` — static assets
